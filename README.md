# ticTacToe

A simple, self-contained Tic Tac Toe game in a single HTML file — no build step, no dependencies.

## Play

Download `index.html` and open it in any browser (or just double-click it).

## Features

- **Two modes** — 2-player hotseat or vs. Computer
- **Unbeatable AI** — the computer uses the minimax algorithm, so it never loses
- **Win/draw detection** with a highlighted winning line
- **Persistent scoreboard** for X, O, and draws across games
